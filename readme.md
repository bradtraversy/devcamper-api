# DevCamper API

> Backend API for the DevCamper application, which is a bootcamp directory app

## Usage

Rename the "config/config.env.env" to "config/config.env" and add your own values/settings

### Install Dependencies

```
npm install
```

### Run app

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
